/* 
Int main - некая функция которая срабатывает при запуске программы 
Return 0 - после окончания функции сигнализирует об окончании программы 
{- тело функции все сто между ними 
; - говорит о том, что строка кода завершена 
%-значение, что помогает выводить переменную 
d - didjest - тоесть цифры, которые мы берём из значения int 
/n - переход на новую строку 
printf - выводит значения на экран
int - целые числа (1, 2, 3 ... 78)
float - числа с запятой, напрмер (2,345678)
scanf - нужна чтобы вводить данные
& - амперсанд, позволяет определить адрес переменной
\ - деление на число, например 123\10=12
% - деление с остатком, например 123%10=3,выводит то что убралось за запятую
""- в них ты вписываешь то, что хочешь видеть на экране, используется в printf
int a,b,c,e,e2,e3,e4,e5,g,g2,g3,g4,g5,a1,b1 - это переменные заданые значения,
с самого начала равняются 0. в любой момент можно их задать (int a=3) 
!= - это не равняется какомуто числу, например: if(a != 3) {код}
For — цикл с предусловием. Это значит, что сначала проверяется условие, 
и если оно true, то дело выполняется.
<=  - меньше или равно заданному числу, в нашем случае 10
++ - это оператор увелечиния значения на 1, а -- наоборот 
<<- сдвиг влево
if-проверка на выполнение действие, например: если число а = 3, то 
возпроизводиться команда а + в 
*/
